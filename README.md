# geofix

1. kalo belum ada venv
```
python3 -m venv venv
```
2. kalo sudah, nyalakan
 ```
 venv/bin/activate
 ```
3. install semua dependency ke venv
```
 install -r requirements
```
4. nyalakan flask
```
flask run 
```
